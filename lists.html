Listas en HTML

En HTML tenemos dos tipos de listas, las ordered List(OL), y las Unordered Lists (UL), estas etiquetas a su vez tiene etiquetas hijas llamadas (li), el uso de una o de otra dependera si el orden en nuestra lista importa

Ejemplo de una lista ordenada (OL)

  1. Elemento 01
  2. Elemento 02
  3. Elemento 03
  4. Elemento 04

Ejemplo de una lista desordenada (UL)

  • Elemento 01
  • Elemento 02
  • Elemento 03
  • Elemento 04
